# NYSC Registration Analysis Dashboard

## Project Overview

This project analyzes NYSC registration data using SQL for data cleaning and Power BI for dashboard development.

## Tools Used

- SQL
- Power BI
- Excel

## Project Process

1. Data collection
2. Data cleaning using SQL
3. Data analysis
4. Dashboard creation in Power BI
5. Insight generation

## Dashboard Highlights

- Total Registrants: 7,491
- Total Institutions: 11
- Average Age: 25 Years
- Analysis of origin states
- Analysis of preferred service states
- Analysis of graduating honors
- Analysis of courses of study

## Key Insights

- Most registrants are between 21 and 30 years old.
- Second Class Upper is the most common graduating honor.
- Some institutions contribute significantly more registrations than others.
- Service state preferences vary across registrants.
